Bootstrap Context Menu

The following table contains the context menu directive, which offers the ability to open up a dropdown menu in case of a click on the table row. The directive is bound via context="targetUL" and references the ul defined in the document. All click events are catched, in case on a document click, the context menu is closed.

The scope is applied to the context menu, and the model is passed through via scope : '@&', so all click events are passed through to the parent controller. That makes the context menu really easy to use and the events easy to catch.
